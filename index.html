fract-ol

Fractals are fascinating, aren't they?

This project is a small exploration of mathematical beauty rendered through code. In fract-ol, I implemented classic fractal sets such as the Mandelbrot and Julia sets, generating them pixel by pixel using iterative complex number calculations.

Zooming, color mapping, and real-time parameter changes helped bring these abstract concepts to life on screen. Through this project, I also strengthened my understanding of graphics programming fundamentals and low-level rendering concepts while improving my overall software development skills.
